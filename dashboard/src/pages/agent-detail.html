Built-in Tools Reference

MQTT

publishMessage(topic, payload)
Publish a message to any MQTT topic.

Data Queries

getTopicValues(topics, archiveGroup?)
Get last known values for comma-separated topics.

findTopics(pattern, archiveGroup?)
Search topics by MQTT wildcard pattern (+ single level, # multi-level).

queryHistory(topic, startTime?, endTime?, limit?, archiveGroup?)
Query archived messages for a topic within a time range (ISO-8601 timestamps).

Agent Notes (Persistent Memory)

saveNote(key, content)
Save a note to persistent memory. Keys can be hierarchical (e.g. decisions/heater).

recallNote(key)
Retrieve a note by its exact key.

searchNotes(pattern)
Find notes by pattern (e.g. # for all, decisions/#).

deleteNote(key)
Delete a note by key.

Notes are stored as retained MQTT messages under agents/{name}/memory/{key} and persist across restarts. Tools with archiveGroup? use the configured default archive group when omitted.
